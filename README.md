> **Easily show character dialogue in any type of game with Visual Scripting!**

**Dialogue Window Node for Unity Visual Scripting** is a free and open-source Unity package is out-of-box highly customizable solution for simple character dialogue in your games. 
This package will help you writing simple character dialogue with scripted situations or branching. 

## ‚ú® Features

- **Effortless Setup**: Drag-and-Drop sample prefab, add entries to Localization Tables, and just place node in your Script Machine.
- **Customizable Animations**: You can add any animations or callback to your window using exposed OnShow and OnClose events.
- **Typewriter Effect**: Provides a typewriter effect to your dialogue to give effect of characters speaking.
- **Ready-to-Use Sample**: Kickstart your creativity with a pre-made dialogue.
- **Rich Text Support**: Apply color, emphasize words with scaling. All Rich Text functionality is supported.

## üì¶ Installation

The easiest way is to [install a package](https://docs.unity3d.com/Manual/upm-ui-giturl.html).

1. Open **Window/Package Manager** in Unity.
2. Click on the **+** button in the top left corner.
3. Select **Add package from git URL** and paste the following URL:

```
https://github.com/KirKami/Dialogue-Window-Node-for-Unity-Visual-Scripting.git
```

## üöÄ Quick Start

1. **Install the package**.
2. Add the **Dialogue Window Component** to your panel or canvas with a dialogue screen in your **starting scene**.
3. Set references to text fields, hint to press interraction and button to continue.
4. Add [Localization](https://docs.unity3d.com/Packages/com.unity.localization@1.5/manual/LocalizationTablesWindow.html) String Tables for your dialogue texts and names of characters who speak.
5. Add **Dialogue Window** nodes to a [Script Machine](https://docs.unity3d.com/Packages/com.unity.visualscripting@1.9/manual/vs-graph-machine-types.html) with your dialogue.
6. See your dialogue in a play mode.
